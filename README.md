  - Thuật toán vẽ đường tròn 
  - Thuật toán vẽ ellipse
2. Thuật toán xén hình và tô màu (Sử dụng sự kiện click chuột)
  + Thuật toán xén hình: 
  - Cohen sutherland
  - Chia nhị phân
  + Thuật toán tô màu: 
  - Tô màu theo đường biên(đệ quy)
3. Phép biến đổi 2D

  - Tỉ lệ với tâm tỉ lệ
